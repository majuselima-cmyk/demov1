Cuan Platform

Overview Flow Tech Stack Architecture

Overview

Deskripsi Platform

Cuan Platform adalah platform staking berbasis blockchain dengan sistem referral dan token reward. Platform ini memungkinkan pengguna untuk melakukan deposit minimal $10 menggunakan USDT, mendapatkan reward staking harian, dan memperoleh keuntungan dari sistem referral.

Fitur Utama

  • Landing Page untuk informasi dan registrasi
  • Dashboard Member untuk mengelola staking dan referral
  • Dashboard Admin untuk manajemen platform
  • Sistem Registrasi dan Login
  • Deposit USDT (Minimum $10)
  • Staking dengan reward 0.5% per hari
  • Sistem Coin dan Token reward
  • Sistem Referral dengan transfer token

Konsep Bisnis

Deposit $10

  • Staking 0.5% per hari dari $10
  • Mendapatkan 90% Coin (coin buatan platform)
  • Mendapatkan 10% Token dari platform
  • Token 10% dapat dicairkan ketika ada referral di bawahnya
  • Token dapat ditransfer ke referral di bawahnya jika sudah deposit

User Flow

Registration Flow

1. User mengakses Register Page
2. Input data: Email, Password, Referral Code (opsional)
3. Validasi input
4. Create account via Supabase Auth
5. Create user profile di database
6. Link referral jika ada referral code
7. Redirect ke Login Page

Login Flow

1. User mengakses Login Page
2. Input credentials: Email, Password
3. Validasi credentials
4. Check user role
5. Redirect ke Dashboard sesuai role (Member/Admin)

Deposit Flow

1. User mengakses Deposit Section di Dashboard
2. Input amount (minimum $10)
3. Select payment method (USDT)
4. Generate payment address/QR
5. Wait for payment confirmation
6. Verify payment (webhook/manual)
7. Process deposit:
- Add $10 ke user balance
- Start Staking (0.5% per hari)
- Allocate 90% sebagai Coins
- Allocate 10% sebagai Tokens
- Check referral chain
- Transfer 10% tokens ke upline jika ada
8. Update Dashboard

Staking Flow

1. Deposit confirmed
2. Start staking process
3. Calculate daily reward: 0.5% dari $10 = $0.05
4. Cron job berjalan setiap hari
5. Untuk setiap active staking:
- Calculate reward
- Allocate 90% ke Coins
- Allocate 10% ke Tokens
- Update user balance
- Check referral chain
- Transfer tokens ke upline jika referral sudah deposit

Referral Flow

1. User A (Referrer) share referral code
2. User B (Referee) register dengan referral code User A
3. Link created: User B.upline = User A.id
4. User B deposit $10
5. Process:
- User B mendapat: 90% Coins, 10% Tokens (locked)
- User A mendapat: 10% Tokens dari deposit User B (withdrawable)
- User B tokens menjadi transferable ke referral mereka

Tech Stack

Frontend

  • Nuxt.js 3 - Full-stack Vue.js framework
  • Tailwind CSS - Utility-first CSS framework
  • TypeScript - Type safety

Backend & Database

  • Supabase - Backend-as-a-Service
  • PostgreSQL - Main database
  • Supabase Auth - Authentication
  • Row Level Security - Data security

Deployment

  • Vercel - Primary deployment platform
  • Cloudflare Pages - Alternative deployment

External Services

  • USDT Payment Gateway - Payment processing
  • Webhook System - Transaction verification

Database Schema

profiles

id (uuid), email (text), referral_code (text, unique), upline_id (uuid), role (text), created_at, updated_at

deposits

id (uuid), user_id (uuid), amount (decimal), currency (text), status (text), transaction_hash (text), created_at, confirmed_at

staking

id (uuid), user_id (uuid), amount (decimal), daily_rate (decimal), status (text), started_at, last_reward_at

user_balances

id (uuid), user_id (uuid), coins_balance (decimal), tokens_balance (decimal), withdrawable_tokens (decimal)

referral_chain

id (uuid), referrer_id (uuid), referee_id (uuid), level (integer), created_at
