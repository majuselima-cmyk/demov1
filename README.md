# Cuan Platform

Platform staking berbasis blockchain dengan sistem referral dan token reward.

## Tech Stack

- **Framework**: Nuxt.js 3
- **Database**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel / Cloudflare Pages
- **Authentication**: Supabase Auth

## Fitur Utama

1. Landing Page
2. Dashboard Member
3. Dashboard Admin
4. Register & Login
5. OTP Verification (Coming Soon)
6. USDT Deposit (Minimum $10)
7. Staking System (0.5% per hari)
8. Coin & Token Reward System
9. Referral System

## Instalasi

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Quick Start

1. Clone repository
2. Install dependencies: `npm install`
3. Setup environment variables (copy `.env.example` to `.env`)
4. Import database schema ke Supabase (`docs/DATABASE_SCHEMA.sql`)
5. Run development: `npm run dev`
